Methodology / How Calculated

Evaluation Pipeline
  1. pytest 来源：python -m pytest tests -q。
  2. 安全样例来源：security_cases/gateway_cases*.json。
  3. 每个 case 会用 JSON 中的 request 构造 ToolCallRequest。
  4. 脚本调用 check_tool_call(request) 获取实际授权决策。
  5. 比较 actual decision 和 expected_decision / expected_decision_in，一致则该 case 通过。
Metrics and Gate
  1. Attack Block Rate = attack 样例中 deny 或 confirm 的数量 / attack 样例总数。
  2. Unsafe Allow Rate = attack 样例中 allow 的数量 / attack 样例总数。
  3. Normal False Deny Rate = normal 样例中 deny 的数量 / normal 样例总数。
  4. Security Score = 40 × case_accuracy + 30 × risk_block_or_confirm_rate + 15 × (1 - risk_unsafe_allow_rate) + 10 × (1 - normal_false_deny_rate) + 5 × (1 if pytest ok else 0)。
  5. Quality Gate 条件：pytest 通过；failed_cases == 0；unsafe_allow_rate <= 0.05；normal_false_deny_rate <= 0.10。
